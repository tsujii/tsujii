*GitHub [#gdbd9925]


GitHub（ギットハブ）はソフトウェア開発プロジェクトのための共有ウェブサービスであり、Gitバージョン管理システムを使用する。
Ruby on RailsおよびErlangで記述されており、GitHub社によって保守されている。

SNSとしての機能も含まれており、公開リポジトリ（誰でも公開できるリポジトリ）で、自分が作ったモジュールやexeファイル等を
共有出来る。

[参考]
http://ja.wikipedia.org/wiki/GitHub
